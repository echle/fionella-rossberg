# Feature: UI-Optimierung

## Ziel
Verbesserung der visuellen Qualität, Reduzierung von Verpixelung und Anpassung der Grundauflösung für ein scharfes, kinderfreundliches Spielerlebnis.

## Motivation
Die aktuelle UI wirkt pixelig, besonders auf größeren oder hochauflösenden Bildschirmen. Ziel ist eine scharfe, responsive Darstellung.

## Anforderungen
- Grundauflösung wird dynamisch und responsiv an die Fenstergröße angepasst (Mindestauflösung für Lesbarkeit, z. B. 800×600)
	- Skalierung erfolgt automatisch, um Verpixelung zu vermeiden
- Phaser-Konfiguration für hohe Auflösung und responsive Skalierung
- Assets und Partikel in passender Qualität
- Tests für visuelle Qualität
- Dokumentation der Maßnahmen

## Akzeptanzkriterien
- Keine sichtbare Verpixelung auf typischen Geräten
- UI passt sich responsiv an verschiedene Bildschirmgrößen an
- Alle Tests bestehen
- Maßnahmen sind dokumentiert
